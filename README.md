# 🔐 SecureVote — Anonymous Voting Web App

A secure and anonymous voting platform built with Firebase.

## ✨ Features

- 🔒 Anonymous voting via SHA-256 hashed UIDs
- 📊 Real-time results
- 🔑 Poll access via unique codes
- 👁️ Observer mode (view without voting)
- 📥 CSV export of results
- 🛡️ reCAPTCHA v3 protection

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Firebase Auth, Firestore
- **Hosting:** Firebase Hosting
